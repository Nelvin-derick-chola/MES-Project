# Mercury Express Logistics

A modern logistics and courier service website built with React, TypeScript, and Vite.

## 🚀 Features

- **Landing Page** with hero carousel and estimate calculator
- **Services Page** showcasing all logistics services
- **Store Locator** with 18+ locations across Zambia
- **Track Shipment** modal with real-time tracking simulation
- **Mobile App Section** with APK download
- **Team Page** featuring company leadership
- **Legal Pages** (Privacy Policy, Terms, Restricted Items, Insurance, Indemnity)
- **Contact Page** with form and location info
- **Fully Responsive** design for all devices
- **Modern Animations** throughout the UI

## 🛠️ Tech Stack

- **React 18** with TypeScript
- **Vite** for fast builds
- **React Router v6** for navigation
- **CSS Modules** for styling
- **Lucide React** for icons
- **React Icons** for social media icons
- **QRCode.react** for QR code generation

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/mercury-express.git

# Navigate to project
cd mercury-express
